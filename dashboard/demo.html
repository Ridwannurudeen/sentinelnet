Section 4

Integration in 3 Lines

Query trust from any environment. Gate smart contracts, automate checks in Python, or curl from the terminal.

from sentinelnet import SentinelNet
sn = SentinelNet()
sn.is_trusted(42)  # Returns True/False
3 lines to trust-gate any agent
import {TrustGate} from "sentinelnet/TrustGate.sol";

function execute(uint256 agentId) external {
    require(gate.isTrusted(agentId));
    // Only trusted agents pass
}
1 line to gate any smart contract function
# Check if an agent is trustworthy
curl https://sentinelnet.gudman.xyz/trust/42

# Response:
# {"trust_score": 82, "verdict": "TRUST", ...}
1 command from any terminal
